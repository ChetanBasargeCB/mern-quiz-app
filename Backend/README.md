# MERN Quiz Web Application

A full-stack quiz application built using the MERN stack.

## Features (Backend)
- User registration and login
- JWT-based authentication
- Secure password hashing
- MongoDB database integration

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT
- Mongoose
